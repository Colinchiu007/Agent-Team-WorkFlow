## 🎯 核心战略决策（已确认）

| 决策 | 内容 |
|------|------|
| 平台优先级 | 微信小程序 → 抖音小程序 → Steam → 网页 |
| 先跑通平台 | **微信小程序**（第一个上线目标）|
| 付费合规 | 抽卡模式需合规设计 |
| 技术选型 | 微信/抖音小程序优先，H5 后续实现，但**设计时预留兼容性** |
| 设计原则 | 架构分层，业务逻辑与平台解耦，方便后续多端适配 |
